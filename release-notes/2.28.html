Release Notes - Jersey 2.28

Release date: 25-January-2019

Jersey 2.28 is the last version of Jersey released from https://github.com/jersey/jersey before it is donated to Eclipse Foundation. From now on, Eclipse Jersey will be located at https://github.com/eclipse-ee4j/jersey Most changes in 2.28 are related to the donation process.

Bug

  • [JERSEY-3828] - SseEventSource#target(null) should throw NPE
  • [JERSEY-3872] - JerseySseBroadcaster#broadcast() returns null
  • [JERSEY-3846] - OsgiRegistry attempts to load classes from comment lines
  • [JERSEY-3871] - Upgrade BV dependencies to latest

Please try out Jersey 2.28 and let us know if you find something which might be improved:

if (exist("https://github.com/eclipse-ee4j/jersey")) {
  Eclipse Jersey
} else {
  Jersey mailing list
  Jersey issue tracker 
}
